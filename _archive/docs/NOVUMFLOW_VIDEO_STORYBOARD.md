## üé® Visual Design Guidelines

### Consistent Elements
- **Brand Colors**: Primary blue (#3b82f6), success green (#10b981)
- **Typography**: Inter font family throughout
- **Spacing**: 16px grid system for all elements
- **Animations**: 0.3s ease-out transitions
- **Icons**: Consistent Lucide React icon set

### Screen Recording Settings
- **Resolution**: 1920x1080 at 60fps
- **Browser**: Clean Chrome window, no extensions visible
- **Cursor**: Large, blue highlight for visibility
- **Timing**: 2-second pause between major actions

### Audio Specifications
- **Voiceover**: Professional, confident, friendly tone
- **Music**: Corporate motivational track at 20% volume
- **Effects**: Subtle UI sounds, no overwhelming audio
- **Quality**: 48kHz, stereo, no background noise
